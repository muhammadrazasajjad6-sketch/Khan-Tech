## 📨 Contact Form Behavior

Instead of traditional backend form handling, the contact form is configured to:

1. Collect user input
2. Generate a custom message
3. Redirect the user to **WhatsApp**
4. Open chat with **Khan Tech**
5. Pre-fill project inquiry details automatically

This makes lead generation faster and easier without requiring a backend server.



You can simply open:

```bash

```
